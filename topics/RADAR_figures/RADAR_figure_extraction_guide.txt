TIPS FOR EXTRACTION:
====================

1. Save images at HIGH RESOLUTION (at least 1200px width)
2. Use descriptive filenames exactly as shown above
3. Save as PNG format for best quality
4. Include any legends, scale bars, or annotations visible
5. Crop to remove excess white space but keep all relevant content
6. If a slide has multiple sub-figures, include all panels

NAMING CONVENTION:
==================
All files must be saved exactly as listed (lowercase, underscores)
Example: em_spectrum.png (NOT Em_Spectrum.png or em-spectrum.png)

DIRECTORY STRUCTURE:
====================
Create directory: RADAR_figures/
Place all 29 PNG files in this directory
